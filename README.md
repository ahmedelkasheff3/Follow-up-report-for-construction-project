# Follow-up-report-for-construction-project
The objective of this project was to develop a progress monitoring report for a residential complex construction project. The report analyzes work progress, illustrates planned versus actual percentages, and evaluates various KPIs, such as Earned Value and the Schedule Performance Index , while proposing corrective actions in the event of delay
